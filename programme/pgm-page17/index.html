关于数组元素的循环遍历

  • 首先我们定义一个用于演示的数组

    //定义数组
    let arr = [];
    for(let i = 0 ; i < 10 ; i++ ){
    arr[i] = i * i ;
    }
    arr.prop = "晨曦洒在我家屋檐上";
    arr.method = function(){
    return "爷爷和我一起去放羊！";
    };

  • JavaScipt数组的forEach遍历

    arr.forEach(function(e){
    // e不是索引，而是每个数组中的值;
    // forEach不能使用break中断循环
    // forEach语句和for...in类似
    console.log(e);
    });

  • JavaScript数组的for...in循环

    for(let index in arr){
    //index是索引值，它是字符串类型
    //for...in 循环会将数组上的属性和方法也遍历出来
    console.log(index + ":" + arr[index]);
    };

  • ES6新方法: JavaScript数组的for...of循环

     for(let turn of arr){
    // for...of循环只遍历数组中具体的值。并不会遍历数组的属性和方法
    // turn不是索引值，而是arr数组中的具体值
    if(turn === 25){continue;}
    console.log(turn);
    };
