jQuery回到顶部小插件(一)

  • 说明：
    • 此插件是基于jQuery插件开发，所以在使用此插件之前请一定要事先引用 jQuery，本插件兼容所有jQuery版本
    • 此插件兼容所有浏览器，如果要兼容IE8及以下，请确保引用的jQuery文件是1.x版本

  • 使用方法

    • 此插件命名为sal_backtop
    • 提供一种属性 ispeed
      • 将ispeed的值设置越大，网页回弹的时间就越长、速度就越慢
      • 如果将ispeed的值设置为数字 0 的话，网页直接回弹到顶部，有设置 锚点#效果相同

    • 使用示例：

      • 不加参数属性的引用：

        $(".obt").sal_backtop();

      • 设置ispeed属性的使用方法：

        $(".obt").sal_backtop({ispeed: 20});

  • 插件代码：

    ;(function($,window,document,undefined){
    "use strict";
    if(!$) { return console.warn('页面没有正确引入jQuery'); };
    $.fn.sal_backtop = function(options){
        var defaults = {
            ispeed :  5 
        }; 
        var settings = $.extend({},defaults,options);
        this.click (function(){
            var timer = setInterval(function(){
                var scrolltop = document.documentElement.scrollTop || document.body.scrollTop; 
                var Ispeed = Math.floor(-scrolltop/settings.ispeed);    
                document.documentElement.scrollTop = scrolltop + Ispeed;
                document.body.scrollTop = scrolltop + Ispeed;
                if(scrolltop === 0){ clearInterval(timer)};
            },30);
        });
        return this;
    };
})(window.jQuery,window,document);
