使元素居中的三种方法

css css3

将一个类名为test的div设置为相对浏览器视口上下左右居中

  1. div.test{
  2.     height: 200px;
  3.     width: 100px;
  4.     background: #ccc;
  5. }

第一种方法

  • 设置样式属性
  1. position: fixed;
  2. top: 0;
  3. left: 0;
  4. bottom: 0;
  5. right: 0;
  6. margin: auto;

第二种方法

  1. position: fixed;
  2. top: 50%;
  3. left: 50%;
  4. margin: -100px 0 0 -50px;

第三种方法

  • 缺点：兼容性差
  1. position: fixed;
  2. top: 50%;
  3. left: 50%;
  4. /* 
  5. *   translate() 方法，元素从其当前位置移动
  6. *   Internet Explorer 9 需要前缀 -ms- 
  7. */
  8. transform: translate(-50%,-50%);
