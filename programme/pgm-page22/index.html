关于form的知识点整理

form 标签

<form action="URL" method="get" accept-charset="utf-8" autocomplete="on" name="form_name" novalidate>

  1. action - 规定当提交表单时向何处发送表单数据
  2. method - 规定用于发送 form-data 的 HTTP 方法
  3. accept-charset - 规定服务器可处理的表单数据字符集
  4. target - 规定在何处打开 action URL (属性值有：_self、 _parent、_top、framename)
  5. autocomplete - HTML5属性
    • 兼容所有主流浏览器
    • firefox和chrome会忽略autocomplete="off"属性，最好当要自动填充的时候在使用autocomplete属性
  6. name - 规定表单的名称
  7. novalidate
    • novalidate 属性是一个布尔属性
    • novalidate 属性规定当提交表单时不对表单数据（输入）进行验证
    • 兼容性：IE10+，不支持Safari，其它浏览器支持

<label for="text_ID" form = "form_name"> &lt;label&gt;标签为 input 元素定义标注（标记）</label><input type="text" name="" value="" id="text_ID">

文本域

  • 单行文本域 input 和多行文本域 textarea 通用属性
    • placeholder
      • HTML5新属性
      • 支持IE11及其它高级浏览器
      • 当value属性和placeholder都设置时，显示value的值
    • readonly
      • readonly 属性规定文本区域为只读
        • 在只读的文本区域中，无法对内容进行修改，但用户可以通过 tab 键切换到该控件，选取或复制其中的内容
      • 所有主流浏览器都支持 readonly 属性
      • readonly 属性是一个布尔属性
    • maxlength
      • maxlength 属性规定文本区域的最大长度（以字符计）
    • required
      • required 属性规定一个文本区域是必需的/必须填写（以提交表单）
      • 支持IE10及其它高级浏览器(safari可能不显示警告框)
      • required 属性是一个布尔属性

单行文本域

<input type="text" name="" value="" placeholder="" >

  • value - 初始文本域中的值

多行文本域

<textarea rows="10" cols="30" autofocus disabled readonly maxlength="100" required wrap="hard" name=""></textarea>

  • 多行文本域
  • 支持所有浏览器
  • rows 和 cols 属性
    • 可以通过 cols 和 rows 属性来规定 textarea 的尺寸大小，不过更好的办法是使用 CSS 的 height 和 width 属性
  • autofocus属性
    • autofocus 属性规定文本区域应该在页面加载时自动获得焦点
    • autofocus 属性是一个布尔属性
    • IE10+及其它高级浏览器支持
    • 此属性其它form标签通用
  • disabled
    • disabled 属性规定文本区域应该被禁用
      • 被禁用的文本区域既不可用，文本也不可选择（不能被复制）
    • disabled 属性是一个布尔属性
    • 支持所有主流浏览器
    • 此属性其它form标签通用
  • wrap
    • wrap 属性规定在表单提交时文本区域中的文本是如何换行的
    • HTML5针对textarea标签的新属性
    • 所有浏览器都支持该属性
    • 属性值
      • soft - 在表单提交时，textarea 中的文本不换行（默认）
      • hard - 在表单提交时，textarea 中的文本换行（包含换行符）
        • 当使用 "hard" 时，必须指定 cols 属性
  • 固定文本域大小
    • 需要在css样式中设置
      • resize 属性
        • 只有IE和其edge不支持该属性
        • resize: none|both|horizontal|vertical;
          • none - 用户无法调整元素的尺寸
          • both - 用户可调整元素的高度和宽度
          • horizontal - 用户可调整元素的宽度
          • vertical - 用户可调整元素的高度

上传文件

<input type="file" name="pic" accept="image/gif, image/jpeg">

  • accept
    • accept 属性只能与 <input type="file"> 配合使用。它规定能够通过文件上传进行提交的文件类型
    • 请避免使用该属性。应该在服务器端验证文件上传
    • 用逗号隔开的 MIME 类型列表

按钮

  • 普通按钮
    • <input type="button" value="按钮"> 等价 <button>按钮</button>
  • 提交按钮
    • <input type="submit" value="提交">
  • 重置按钮
    • <input type="reset" value="重置">

fieldset-legend 分组

 <fieldset>
	    <legend>标题或说明标题</legend>
	    <input type="button" name="" value="">

标题或说明标题
  • <fieldset> 标签可以将表单内的相关元素分组
  • <legend> 标签为 <fieldset> 元素定义标题
  • 兼容所有浏览器
