JavaScript正则表达式

  • 对字符进行匹配的方法
    • 作用
      • 匹配字符
    • 工作原理
      • 通配符匹配技术
  • js中有正则regular expression的内置对象
    • var re = new RegExp("正则的内容","正则的标志")
    • 另一种 var re = /正则的内容/正则的标志
      • eq： var re = /a/i
        • 匹配字符中的所有a，加上i的意思是不区分大小写
        • *匹配所有字符
        • \转义

  • 特殊符号都需要转义

    • 特殊符号
      • * ! $ ^ * + = | . ? \ / ( ) [ ] { }
    • 特殊字符
      • o t n v f r xnn uxxx cX

  • 字符类

    • . 匹配任意字符(除换行符以外:\n,\r,\u2028或者\u2029)
    • \w 匹配任何ASCII单字符[a-z A-Z 0-9 _]
    • \W 匹配ASCII单字符之外的字符
    • \s 匹配任何Unicode空白符（就是空格）
    • \S 匹配任何Unicode空白符之外的字符（就是非空格）
    • \d 匹配任何数字[0-9]
    • \D 匹配任何数字之外的字符
    • [...] 匹配方括号中的所有字符
    • [^...] 匹配非方括号中的所有字符
      • 当^在正则表达式的最前一个字符时，表示匹配字符串的开头。当^在【】中时，表示“非”，也就是“取反”的意思。

  • 重复类

    • {n} 匹配前一项n次
      • eq: var re = /a{3}/i
        • 需要连续三个字符a出现 如：aaabcd
    • n, 匹配前一项至少一次，至多不限
    • {n,m} 匹配前一项至少n次，最多m次
    • {?} 匹配前一项0次或者1次
    • {+} 匹配前一项至少1次
      • eq:x+?
    • {*} 匹配前一项0次或者多次

  • 选择符 |

    • 分隔可以用来选择的文字
    • eq: `var re = /a|b/i
      • 匹配a字符或者b字符

  • 定位符

    • ^ 匹配字符串的开头
      • eq: var re = /^a/i
        • a必须出现在字符串的开头
    • $ 匹配字符串的结尾
      • eq: var re = /a$/i
        • a必须在行的末尾
    • \b 匹配一个单词的边界
    • \B 与\b相反，匹配一个非单词边界

  • 分组

    • (...) 将几个项目组合成一个单元
    • \n 表示使用分组符(x)匹配到的字符串
      • eq: /(abc)\1/.test('abcabc');
    • (?...)与(...)类似，只是不记忆与该组合所匹配的内容

  • 标志

    • i 匹配时不区分大小写
      • eq: /abc/gim.test("AB") - g,i,m无所谓顺序
    • g 匹配时执行全局匹配
    • m 匹配时执行多行匹配

  • 正则表达式对象的方法

    • exec() 执行正则表达式的匹配，返回一个数组
    • test() 测试正则表达式的匹配，返回布尔值
    • toSource() 返回RegExp对象的源代码
    • toString() 将RegExp对象转成字符串
    • compile()
