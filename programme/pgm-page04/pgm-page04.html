BOM视口信息获取相关API

  • 获取浏览器窗口尺寸（浏览器的视口，不包括工具栏和滚动条）

    • 第一种：

      // 兼容性：支持ff，chrome，IE9及以上
window.innerHeight
window.innerWidth  //包括滚动条宽度

    • 第二种：

      // 兼容性： 支持所有高级浏览器
document.documentElement.clientHeight
document.documentElement.clientWidth

  • 获取body的高度和宽度

    /* 兼容性：所有浏览器 */
    //第一种
    document.body.clientHeight
    document.body.clientWidth
    //第二种
    document.body.offsetWidth
    document.body.offsetHeight
    //第三种
    document.body.scrollWidth
    document.body.scrollHeight
/* 三者使用效果相同 */

  • 获取访问者屏幕的宽度和高度的方法

    //俩种方法全部兼容所有浏览器（window对象可省略）
/* 第一种 */
screen.availWidth 
screen.availHeight 
/* 第二种 */
window.screen.height 
window.screen.width

  • 网页被卷去内容的上部高度和左部高度：

    // 兼容所有IE、Firefox：
document.documentElement.scrollTop;
document.documentElement.scrollLeft;
//兼容chrome、Opera：
document.body.scrollTop;
document.body.scrollLeft;
//可以这么写
var scrollHeight = document.documentElement.scrollTop || document.body.scrollTop;

  • 获取元素的高度和宽度

    // 兼容所有浏览器
 var dragW = drag.offsetWidth; //元素的可见宽度
 var dragH = drag.offsetHeight; //元素的可见高度

  • 没用的属性

    • 网页正文部分上：window.screenTop
    • 网页正文部分左：window.screenLeft
    • 在Firefox为undifined，在Chrome 和 IE显示结果也并不一致
