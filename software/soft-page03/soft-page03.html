Windows 系统常用快捷键

  • 说明

    • WIndows的快捷键很多，而且各个Windows版本的系统的快捷键也大同小异，这里我只介绍Windows 7旗舰版系统的一些比较常用的快捷键操作。

  • 与WIN键相关的快捷键

    1. Win键 + 方向键：win+ ↑ 将窗口最大化；win+ ↓将窗口最小化；←、→方向分别是将窗口靠到桌面的左右侧，并将窗口自动对半占据。

    2.Win键 + Space空格键：可以将当前的窗口透明化，松手之后就还原了，相当于快速显示桌面预览。

    1. Win键 + Tab键：打开3D任务切换功能，当前的任务窗口就成3D形式浮现在屏幕上，通过不断的点击Tab键就可以切换到你想要的切口.(此功能XP系统并不具备。)

    2. 与Win + Tab有相似功能的快捷键组合还有Ctrl + Alt + Tab与Ctrl + Win + Tab。

    3. Win键 + X：打开计算机移动中心

    4. Win键 + E：打开计算机

    5. Win键 + (+/-)：打开Windows放大、缩小功能

    6. Win键 + D:换到桌面

    7. Win键 + L:锁定计算机

    8. Win键 + R:打开运行对话框

    9. Win键 + B:切换到通知栏

  • 与Ctrl相关的快捷键

    1. Ctrl + A:全选；Ctrl + C:复制；Ctrl + V：粘贴；Ctrl +　V剪贴

    2. Ctrl + Tab：在选项卡中切换

    3. Ctrl + Z：撤销操作

    4. Ctrl + F4：关闭当前活动文档

    5. Ctrl + 鼠标滚轮 更改桌面上的图标大小

    6. 在浏览器中：

      • ctrl + shift + delete 清除浏览器历史
      • ctrl + r 刷新浏览器网页
      • ctrl + t 打开一个新标签页
      • ctrl + n 打开一个新窗口
      • Ctrl + h 打开浏览器历史记录

    7. Ctrl + Shift 加某个方向键键：选择一块文本

    8. Ctrl + Shift + N：新建文件夹

  • 在运行中快速打开程序的快捷键

    1. 快速打开Miscrosoft Word的命令：winword

    2. psr.exe: 微软自带录屏快捷键

    3. devmgmt.msc：打开设备管理器控制单元命令

    4. wf.msc：打开高级安全Windows防火墙

    5. diskmgmt.msc：打开磁盘管理器

    6. compmgmt.msc：打开计算机管理

    7. lusrmgr.msc：打开用户管理程序

    8. fsmgmt.msc：打开共享文件夹管理程序

    9. taskschd.msc：打开任务计划程序

    10. services.msc：打开服务管理程序

    11. perfmon.msc：打开性能监视器程序

    12. gpedit.msc：本地组策略编辑器程序
