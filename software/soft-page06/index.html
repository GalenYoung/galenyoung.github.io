Sublime Text 3环境配置

  1. 软件安装包下载地址： http://www.sublimetext.com/3

  2. 常用插件配置：

    1. package control：方便其它插件的快速安装
    2. EMMET： 快速生成HTML结构，并且快速输入HTML代码
    3. IMESupport：解决输入法显示位置异常的问题
    4. autoFileName：要引入文件和地址时，出现提示
    5. LESS ：使LESS代码有颜色
    6. SideBarEnhancements：可以设置打开各个浏览器的快捷键
    7. DocBlockr： 注释插件
    8. SublimeCodeIntel：代码提示插件
    9. Bracket Highlighter: 使代码高亮显示
    10. python 3: 写python代码
    11. HTML5
    12. jQuery

  3. package control插件管理器安装方法：

    • 菜单view > Show Console 调出命令行工具粘贴命令
    • 将（import urllib.request,os,hashlib; h = '2915d1851351e5ee549c20394736b442' + '8bc59f460fa1548d1514676163dafc88'; pf = 'Package Control.sublime-package'; ipp = sublime.installed_packages_path(); urllib.request.install_opener( urllib.request.build_opener( urllib.request.ProxyHandler()) ); by = urllib.request.urlopen( 'http://packagecontrol.io/' + pf.replace(' ', '%20')).read(); dh = hashlib.sha256(by).hexdigest(); print('Error validating download (got %s instead of %s), please try manual install' % (dh, h)) if dh != h else open(os.path.join( ipp, pf), 'wb' ).write(by) ）括号里面的内容粘贴进去。
    • 回车，重启Sublime Text 3即可。
    • 检查插件管理安装成功：如果Preferences > Package Control 这个选项，即表示插件管理器安装成功！

  4. 其它插件的安装方法类似：用ctrl + shift + p 调出面板选择 install package安装插件即可。

  5. SideBarEnhancements配置：

    • 选择以下菜单选项 preferences / Package Settings / Side Bar / Key Bindings - User

    • 将以下代码复制进去：

      [
    { "keys": ["ctrl+shift+c"], "command": "copy_path" },
    //chrome
    { "keys": ["f12"], "command": "side_bar_files_open_with",
        "args": {
            "paths": [],
            "application":"C:\\Users\\Administrator\\AppData\\Local\\Google\\Chrome\\Application\\chrome.exe",
            "extensions":".*" //匹配任何文件类型
        }
    },
    //firefox
    { "keys": ["shift+f12"], "command": "side_bar_files_open_with",
        "args": {
            "paths": [],
            "application":"C:\\Program Files\\Firefox Developer Edition\\firefox.exe",
            "extensions":".*"
        }
     },
    //ie
    { "keys": ["ctrl+f12"], "command": "side_bar_files_open_with",
        "args": {
            "paths": [],
            "application":"C:\\Program Files\\Internet Explorer\\iexplore.exe",
            "extensions":".*"
        }
    },
]
      • chrome打开快捷键：F12
      • firefox打开快捷键：shift + F12
      • IE打开快捷键：ctrl + F12

    • 注意文件的路径引用，每个电脑的浏览器引用路径并不一定相同，而且间隔符号要用双斜杠 \。
